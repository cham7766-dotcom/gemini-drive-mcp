# 사용법 가이드 📖

## 기본 실행
```bash
python src/main.py
```

## 대화형 모드
```
💡 요청: Python으로 Hello World 만들어줘
[AI가 코드 생성]

💡 요청: exit
[프로그램 종료]
```

## 명령어
- **코드 생성**: 자연어로 요청
- **종료**: "exit" 또는 "종료"

## 예시
```
💡 요청: Python으로 CSV 파일 읽고 분석하는 코드
💡 요청: matplotlib으로 그래프 그리는 코드
```

## 문제 해결
- API 키 오류: config/.env 확인
- 패키지 오류: pip install -r requirements.txt 재실행
- 로그 확인: logs/app.log 참조
